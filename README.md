```
USER_ID_SECRET_KEY=
WEAVIATE_API_KEY=
WEAVIATE_REST_URL=
UPLOADTHING_TOKEN=
TRIGGER_SECRET_KEY=
```

## Trigger.dev setup

1. Generate an api key and set environment variable.
2. Put in trigger.dev all the environment variables needed.
3. Run `npx trigger.dev@latest deploy` to deploy all triggers.
